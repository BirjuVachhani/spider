## How to check for updates

Run this command to check for any new updates. If a new version is available on [pub.dev](https://pub.dev/packages/spider)
then it will be shown.

```shell
spider --check-for-updates
```

## How to see tool license

This command prints the license of this tool. See full license [here](license.md).

```shell
spider --license
```

## About

This command prints tool information such as current version, author, etc.

```shell
spider --about
```

## How to open docs

This command will open this documentation in your browser.

```shell
spider --docs
```
